# Archive

This folder retains superseded v2 governance, skill-pack, and legacy test surfaces that no longer define the canonical runtime product.

- `legacy_v2/`: former `avn_v2` runtime, skill-pack, v2 configs, v2 data bundles, and v2 docs.
- `legacy_runtime/`: former non-canonical `avn` analytical/runtime modules removed from the live package surface during consolidation cleanup.
- `legacy_tests/`: historical tests tied to the archived surfaces.
